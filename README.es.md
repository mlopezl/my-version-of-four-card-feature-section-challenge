# Frontend Mentor – Solución al reto Four Card Feature Section

Esta es mi solución al reto **Four Card Feature Section** de [Frontend Mentor](https://www.frontendmentor.io/).  
Este desafío me ayudó a mejorar mis habilidades en **HTML y CSS** al construir una sección limpia y responsiva que muestra cuatro tarjetas de funcionalidades con un diseño moderno.

## Descripción General
Este proyecto es una **sección de funcionalidades responsiva** que muestra cuatro tarjetas, cada una representando un elemento clave de un sistema de gestión de proyectos impulsado por inteligencia artificial.  
Las tarjetas destacan las ideas de **supervisión, creación de equipos, karma y cálculo**.  
El diseño se adapta a diferentes tamaños de pantalla, manteniendo el balance visual mediante un diseño responsivo y componentes limpios.

## El Desafío
Los usuarios deben ser capaces de:

- Visualizar correctamente el diseño en distintos tamaños de pantalla.  
- Ver las cuatro tarjetas organizadas de forma estructurada.  
- Leer cada característica junto con su ícono y descripción.  
- Percibir sombras sutiles y una apariencia moderna en las tarjetas.

## Mi Proceso
1. Estructuré el marcado utilizando **HTML semántico** (`header`, `main`, `section`, `article`, `hgroup`) para asegurar claridad y accesibilidad.  
2. Apliqué **Flexbox** para alinear las tarjetas y crear el diseño de tres columnas (con dos tarjetas apiladas en la columna central).  
3. Configuré **variables personalizadas en CSS** para gestionar la paleta de colores de forma consistente.  
4. Utilicé la **metodología BEM** para mantener los nombres de clases escalables y claros.  
5. Diseñé **tarjetas estilo UI** con sombras, padding y bordes superiores de color para lograr una apariencia moderna.  
6. Implementé **puntos de quiebre (media queries)** para ajustar el orden de las tarjetas y la tipografía en tablet y dispositivos móviles.  
7. Presté especial atención al espaciado y la legibilidad para mantener una interfaz limpia y minimalista.

## Tecnologías Utilizadas
- HTML5  
- CSS3  
- Flexbox  
- Variables Personalizadas de CSS  

## Lo que Aprendí
- Cómo aplicar **HTML semántico** para mejorar la estructura y la accesibilidad.  
- Cómo construir un diseño **basado en tarjetas utilizando Flexbox**.  
- Cómo organizar estilos con la **metodología BEM** para mantener escalabilidad.  
- Cómo mantener un **sistema global de colores usando variables de CSS**.  
- Cómo estructurar **cambios de diseño responsivo** mediante media queries.  
- Cómo mantener el balance visual al diseñar para pantallas pequeñas.
